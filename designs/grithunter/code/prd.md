# GritHunter: Product Requirements & Mission

## Problem
In 2026, "AI-slop" has broken recruiting. LLMs make mediocre engineers look like seniors on paper.

## Solution
GritHunter is a High-Precision Talent Weapon for CTOs to find the 1% of engineers with "Grit"—the ability to solve complex, stateful, and architectural problems.

## User Journey
1. **Hydration (Architect Agent):** High-fidelity technical goal definition via chat.
2. **Hunting:** Scouring GitHub, Kaggle, HuggingFace for "Smoking Gun" PRs.
3. **Forensic Audit:** Ruthless audit of raw `.diff` for "Edge Case Density" and "Architectural Intent."
4. **The Reveal:** A dashboard of traceable code markers and hire/skip recommendations.

## Design Vibe: "Traceable Intelligence"
- **Target Audience:** Skeptical CTOs.
- **Visual Style:** Data-backed truth, not marketing fluff. Surgical, authoritative, fast.
- **Key Feature:** The "Diligence Report" and "Grit Markers."
